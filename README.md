# arduino-practice

Practice repository for Arduino exercises and exam prep.

## Structure
- `sketches/`: small focused programs
- `lib/`: reusable helper libraries
- `test/`: host-side tests or simulation helpers
- `docs/`: notes, wiring diagrams, troubleshooting
- `assets/`: images and circuit captures

## Quick Start
1. Open Arduino IDE.
2. Open one folder under `sketches/`.
3. Select board and port.
4. Verify and upload.

## Learning Path
1. Blink and timing
2. Button input and debouncing
3. PWM and LED effects
4. Multiple LEDs and state machine
5. Sensor + actuator mini projects
